# Project Charter

**Project Name:** Pacific Northwest Hiking Web App  
**Project Sponsor:** The Great Outdoors, Evan Holt  

## 1. Project Purpose / Background
To build a modern, centralized and user friendly web application for PNW hikers that provides reliable trail info and educates users about ‘leave no trace’, safety and actively promotes lesser known trails to help reduce overcrowding.

## 2. Project Objectives
* **Objective 1:** Provide a list of hiking trails
* **Objective 2:** User friendly interface
* **Objective 3:** Display basic trail information (descent, distance, elevation gain)
* **Objective 4:** Educates users on how to ‘leave no trace’ and on env. stewardship

**Success Criteria:**
* List of trails is available to users
* The website loads correctly
* User can navigate easily
* Information available on env. stewardship and ‘leave no trace’

## 3. Scope
**In Scope:**
* MVP
* User accounts and passwords *(Note: Ensure this aligns with MVP assumptions below)*
* Reviews and ratings
* Simple navigation
* Trail details

**Out of Scope:**
* Payment features
* Login system *(Note: Reconcile this with "In Scope" accounts for final client approval)*
* Advanced Maps
* Accounts Administration
* Database info

## 4. High-Level Deliverables
* **Deliverable 1:** Trail info and details
* **Deliverable 2:** MVP features implemented
* **Deliverable 3:** Basic Ui Design
